# Burger Brothers Berlin

Burger Brothers Berlin için Next.js tabanlı sipariş, admin, TV ve takip sistemi.

## Bereiche

- Landingpage
- Menü
- Warenkorb
- Checkout
- Adminbereich
- TV-Monitor
- Bestellverfolgung
- Kundenverwaltung
- Gutscheine
- Kampagnen
- Einstellungen
- Fahrerbereich
- Druck-/Print-Anbindung

## Technik

- Next.js App Router
- TypeScript
- TailwindCSS
- Prisma
- PostgreSQL
- Lokaler Cache nur als Fallback
- Datenquelle für Bestellungen, Produkte, Einstellungen und Kunden: Datenbank

## Lokale Entwicklung

```bash
npm install
npm run dev
